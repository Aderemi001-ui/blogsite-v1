The application is a user-centric blog website that allows users to create and view blog posts on a variety of topics that serves as a plethora to provide information to its users ...
It uses the concept of routing using express to constantly navigate between pages...It also uses body-parser tom parse items from frontend to backend or from database to backend...
I also employed the use of ejs package that gives the avenue for parsing backend parameters into the frontend as the files are named ejs files like the package name...
It also gives admin control to the creator and those the creator wants to be admins
